## 🎯 Prompt 1: AI Image Generator (Midjourney / DALL-E / Firefly)

**Tiếng Anh** (AI tool nhận tiếng Anh tốt hơn):

```
A premium vertical roll-up standee design poster for "8FM Pickleball"
sports community club, 60x160cm portrait format (aspect ratio 3:8).

Style: Modern sports brand aesthetic, vibrant emerald green (#1D9E75)
and amber gold (#F59E0B) color palette, white background with subtle
geometric patterns, premium typography, energetic and youthful feel —
inspired by Nike Run Club and Under Armour event branding.

Layout (top to bottom):
- Top 15%: Brand bar with circular logo placeholder on left, bold
  modern sans-serif wordmark "8FM PICKLEBALL" with tagline "CỘNG ĐỒNG
  PICKLEBALL"
- Hero 30%: Dynamic action illustration of pickleball paddle hitting a
  ball, with motion lines, splashes of emerald and amber, on soft
  gradient mint background
- Headline 18%: Large bold Vietnamese text "QUÉT QR ĐÁNH NGAY" in
  white on emerald gradient block, with amber accent badge "⚡ 15 GIÂY"
- QR section 25%: Large white square area (22x22cm) for QR code with
  rounded border, accompanied by 3 vertical benefit icons with labels:
  "🎯 TÍCH ĐIỂM / 🎁 ĐỔI QUÀ / 🏆 GIẢI ĐẤU"
- Offer strip 5%: Amber gold band with text "🎁 ĐĂNG KÝ MEMBER NHẬN +20 ĐIỂM"
- Footer 7%: Dark emerald with URL and contact info

Typography: Inter font family, bold weights 700-900 for headlines.
Mood: Professional, vibrant, inviting, urgent call-to-action.
High resolution print quality, 300 DPI, CMYK color space.

--ar 3:8 --v 6 --style raw --no people faces text-heavy-design
```

**Tham số:** `--ar 3:8` cho Midjourney (60:160 = 3:8). DALL-E dùng
"tall portrait aspect ratio". Firefly có preset "Vertical Poster".

## 🎨 Prompt 2: Canva AI / Figma AI / ChatGPT (có vision)


## 📚 Tips dùng prompt

### Khi prompt cho AI image:
- **Không generate QR code bằng AI** — không scan được. Tạo placeholder
  trắng → paste QR thật sau (dùng [qrserver.com](https://api.qrserver.com))
- **Tránh nhồi nhét text** — AI generate text thường lệch ký tự tiếng
  Việt có dấu. Tốt nhất: AI vẽ visual, anh add text bằng Figma/Canva sau
- **Reference image**: upload file `walkin-standee.html` screenshot làm
  reference để AI hiểu layout

### Khi gửi designer:
- Gửi kèm logo PNG (anh có sẵn `/icon-192.png`)
- Gửi screenshot file HTML standee có sẵn làm reference
- Gửi brand color hex codes (đã list ở trên)
- Yêu cầu **3 concept** rồi chọn 1 → revise

### Khi tự design trên Canva/Figma:
- Canva: search "Sports event poster vertical" → chọn template gần
  nhất → edit color + text
- Figma: tạo frame 600×1600px → dùng cùng brand kit từ web app
- Export PDF cuối cùng → in
